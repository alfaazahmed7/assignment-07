# 🚀 KeenKeeper

## 📌 Overview

**KeenKeeper** is a modern relationship & contact management web app that helps you stay connected with the people who matter most. It visually tracks your interactions, reminds you when it’s time to reconnect, and gives you meaningful insights into your social life.

---

## ✨ Description

KeenKeeper is built to solve a simple but real problem: *we forget to stay in touch*.
With a clean UI and smart tracking system, it helps you manage friendships, monitor communication gaps, and maintain stronger relationships effortlessly.

---

## 🛠️ Technologies Used

* **Next.js (App Router)**
* **React**
* **Tailwind CSS**
* **Context API**
* **React Toastify**
* **Next/Image Optimization**

---

## 🔥 Key Features

### 1. 📊 Smart Relationship Tracking

Track how long it’s been since you last contacted someone and categorize them with statuses like:

* Overdue
* Almost Due
* On-Track

---

### 2. 👥 Dynamic Friends Dashboard

A clean, responsive grid of friend cards showing:

* Profile image
* Tags (e.g., Close Friend, Work, College)
* Last interaction time
* Current status

---

### 3. ⚡ Smooth UX with Modern Loading

* Suspense-based loading
* Skeleton/grid loaders
* Smooth hover animations
* Optimized image loading

---

## 💡 Why KeenKeeper?

Because relationships shouldn’t fade due to forgetfulness.
KeenKeeper helps you stay intentional, consistent, and connected.
